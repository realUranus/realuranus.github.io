Java 内部类

简介

访问权限public private protected

内部类也可以用访问控制符来修饰。

public：任何地方都可以访问
private：只能在外部类访问
protected：同一个包 或者 继承了外部类
不加权限控制符（default）：同一个包可以访问（不同包继承了也不行，提示：Cannot accessed from outside package）
