【Android】Intent

官方地址

https://developer.android.google.cn/reference/android/content/Intent

简介

An intent is an abstract description of an operation to be performed.

是一种操作描述。

Its most significant use is in the launching of activities, where it can be thought of as the glue between activities.

最重要的用处是：启动activity

构造 Structure

解析 Resolution

标准Activity动作

最重要使用最频繁的是：ACTION_MAIN_和ACTION___EDIT

标准Broadcast动作

数据大小限制

大小：1MB

原因：Intent传递数据的时候，使用到了Binder。

The Binder transaction buffer has a limited fixed size, currently 1Mb, which is shared by all transactions in progress for the process.

解决方式：EventBus

【Android】Activity

官方地址

https://developer.android.google.cn/guide/components/activities/intro-activities

生命周期Lifecycle

https://developer.android.google.cn/guide/components/activities/activity-lifecycle

onCreate

State: Created
Work: perform basic application startup logic that should happen only once for the entire life of the activity.

onStart

State: *Started 可见，为可交互做准备
Work: makes the activity visible to the user**

onResume

State: Resumed 可交互的
Work: This is the state in which the app interacts with the user

onPause

State: Paused
it indicates that the activity is no longer in the foreground 不在前台，不代表被销毁了。

onStop

State: Stopped

no longer visible to the user
